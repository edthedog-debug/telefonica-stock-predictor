## 🎯 What is this?

A **hybrid financial engineering model** for predicting **Telefónica (TEF.MC)** stock prices using:

- 🎲 **Monte Carlo Simulation** (Geometric Brownian Motion)
- 📊 **Technical Analysis** (SMA, RSI, Moving Averages)
- 🤖 **Automated Daily Updates** with REAL market data
- 📈 **Interactive Visualization** with Chart.js
